用Markdown来渲染微信公众号排版

Wednesday, Jun 3, 2020

每次发布公众号可以节约大概30-60分钟。

工具

  1. www.mdnice.com
  2. Chrome Markdown Nice 插件

写作流程

  1. 完整的Markdown语法写到github博客中
  2. 用PicGo插件粘贴图片到自己的图床
  3. 拷贝一份到公众号后台
  4. 自动排版
  5. 添加头部和尾部
  6. 自动转换外链
  7. 发布
