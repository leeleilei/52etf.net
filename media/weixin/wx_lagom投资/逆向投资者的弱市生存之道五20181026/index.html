珍惜熊市下各类福利~

今年行情很差，券商营业部大多这样~

而当初大牛市，曾经是这样的

旺季 >> 卖方市场 >> 服务费涨价淡季 >> 买方市场 >> 服务费打折普世的市场定价规律，咱小散参与的股市也如此。

笔者最近翻出一沓早年的交割单，胸闷~

当初年少无知，不知低交易成本的意义。

另：切莫自认是老客户 = 享受低佣&低费率！
